# Healthcare Financial Reconciliation Platform

## Overview

The Healthcare Financial Reconciliation Platform is a Python-based backend application developed during my internship at Sir Ganga Ram Hospital. The project was designed to automate financial reconciliation by comparing transaction records across multiple enterprise systems and identifying discrepancies.

## Problem Statement

Manual reconciliation of financial records across different systems is time-consuming and prone to human error. This project streamlined the process by validating transaction data and highlighting inconsistencies for faster review.

## Key Features

- Automated reconciliation across multiple enterprise systems
- Receipt and transaction validation
- Detection of missing records and amount mismatches
- Interactive dashboard for reconciliation analysis
- SQL-based data processing
- Oracle Fusion REST API integration

## Technologies Used

- Python
- Streamlit
- SQL Server
- Oracle Fusion REST APIs
